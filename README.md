# Simple Interest Calculator

This project contains a simple interest calculator script.

## Formula

Simple Interest = (Principal × Rate × Time) / 100

## Features

- Calculates simple interest
- Accepts principal amount
- Accepts annual rate of interest
- Accepts time period

## Usage

Run the bash script:

```bash
bash simple-interest.sh
```

Enter:
- Principal amount
- Rate of interest
- Time period

The program will display the calculated simple interest.
